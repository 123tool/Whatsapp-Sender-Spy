**SPY WA SENDER** adalah framework otomasi WhatsApp tingkat lanjut. Dirancang untuk keperluan marketing dan layanan pelanggan otomatis tanpa biaya API resmi yang mahal.

## ⚡ Fitur
- **Massive Blast:** Kirim ribuan pesan dari list `numbers.txt`.
- **Smart Auto-Reply:** Membalas pesan masuk secara otomatis berdasarkan keyword.
- **Anti-Ban Algorithm:** Jeda waktu acak (Random Delay) untuk meniru perilaku manusia.
- **Session Manager:** Tidak perlu scan QR berulang kali.
- **Media Support:** Kirim gambar, dokumen, atau video secara otomatis.

## 📦 Instalasi

### 💻 Windows (CMD) / Linux
1. Install [Node.js](https://nodejs.org/).
2. Clone repository:
```bash
git clone https://github.com/123tool/Whatsapp-Sender-Spy.git
cd Whatsapp-Sender-Spy
npm install
```
### Jalankan : node spy-wa.js
​📱 Termux (Android)
```bash
pkg update && pkg upgrade
pkg install nodejs git chromium -y
git clone https://github.com/123tool/Whatsapp-Sender-Spy.git
cd Whatsapp-Sender-Spy
npm install
node spy-wa.js
```
## Cara Pakai
​Blast: Isi nomor target di numbers.txt (satu nomor per baris, gunakan format 628xxx).
​Auto-Reply: Edit bagian autoReplyConfig di dalam file spy-wa.js untuk menyesuaikan jawaban bot.
​Scan : 
```bash
Scan QR Code yang muncul di terminal menggunakan WhatsApp Anda.
```
